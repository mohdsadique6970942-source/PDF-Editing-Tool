# PDF-Editing-Tool
PDFMania is a lightweight, all-in-one PDF editing tool for quick and efficient document management. It allows users to create, merge, split, compress PDFs, add watermarks, remove pages, and insert custom text. Designed for simplicity and speed, it delivers powerful PDF operations directly in your browser.

---

## 🚀 Features

- 📄 Create PDF  
- 🔗 Merge PDFs  
- ✂️ Split PDF  
- ❌ Remove pages  
- 📝 Add text to PDF  
- 💧 Add watermark  
- 📉 Compress PDF
